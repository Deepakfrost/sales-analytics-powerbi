# 📊 Sales Analytics Dashboard with Power BI

This project is an end-to-end sales analytics dashboard built using **Power BI**, with data sourced from a **SQLite database** and prepared using **Python**.

It provides interactive insights into sales trends, customer segments, top products, city-wise performance, and revenue analysis.

---

## 🧰 Tools Used

- **Power BI** – for data visualization and dashboard creation  
- **SQLite** – lightweight relational database for storing raw sales data  
- **Python (Pandas)** – for data cleaning, merging, and CSV export  
- **GitHub** – for version control and portfolio showcase

---

## 📁 Project Structure

sales-analytics-powerbi/
├── sales_dashboard.pbix # Power BI dashboard file
├── sales_analysis.csv # Cleaned and enriched data used in Power BI
├── README.md # Project documentation (this file)
└── sales.db # Original SQLite database (optional)

## 📊 Dashboard Highlights

- ✅ Total revenue overview (KPI card)
- 📆 Monthly sales trends (Line chart)
- 🛍️ Top 10 products by revenue
- 🧍 Customer segment performance
- 🌆 Revenue by city (bar chart or map)
- 📅 Date slicers and segment filters for interactivity
